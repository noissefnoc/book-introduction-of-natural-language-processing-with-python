### 第一部 データを準備しよう
#### 1章 実行環境を整えよう

本書はWindows 10のWindows Subsystem for Linux上でのUbuntuに


* Python 3：プログラミング言語
* Solr：検索エンジン
* CaboCha：構文解析

という構成。当方の環境はmacOSなので、適宜読み替えを実施。

また、以下の方針で前提・方針で進める。

* 開発環境
    * PyCharm
* コードレイアウト
    * 一旦本誌に従うが、dockerイメージ作成時などに一部変更の可能性あり
* バージョン管理
    *  git
        * 節程度の進捗でコミット
        * 章の完了でタグを打つ
* 動作環境
    * docker
    * Python Web App
    * Solr：dockerの公式イメージ
    * CaboCha：Python Web Appと同居か別かは後の章で決める
